jω processor

A SaxonJS/JavaScript processor of Invisible XML

This processor runs entirely within the browser client, using SaxonJS as the top-level program and the jwiXML JavaScript library. There is no server-side processing, apart from initial delivery of necessary files.

There are two sections - some predefined test examples and an interactive iXML processor. Click on the labels to expand that section

In a couple of days I'll add a Help section ;-), but if you've been following other tutorial material, such as Steven Pemberton's, the controls should be self-explanatory.
