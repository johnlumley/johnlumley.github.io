QT4 XPath 4 equivalent functions

These are available XSLT 3 functions that emulate new or modified XPath 4 functions. The links point to the specification definition.

The namespace prefix mappings xmlns:f="MyFunctions" xmlns:ar="MyArray" xmlns:ma="MyMap" should already be defined in the example stylesheets

Calls can only be made my positional arguments - keyword binding is not supported.

  • ar:empty()
  • ar:foot()
  • ar:index-of()
  • ar:index-where()
  • ar:members()
  • ar:of-members()
  • ar:trunk()
